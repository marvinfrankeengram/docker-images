# Jenkins-Slave

This is another remix of existing docker images to easily run jenkins slaves.

Inspired by evarga/docker-images and mzagar/jenkins-slave-jdk-maven-git

- FROM ubuntu:disco
- git
- Open JDK 8
- python3 & pip
- aws-cli
- ECR credentials helper
- jq
- Maven 3.6.3
- Jenkins user
- SSH access
